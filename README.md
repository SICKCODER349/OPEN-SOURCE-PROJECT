# 🖥️ Open Source Software Capstone Project

## 📌 Project Title

**Open Source System Audit using Shell Scripts**

---

## 📖 Description

This project demonstrates the practical implementation of open-source concepts using Linux shell scripting. It focuses on automating system-level tasks such as system monitoring, package inspection, disk auditing, log analysis, backup creation, and process management using Bash on Ubuntu (WSL).

---

## 🎯 Objectives

* Understand open-source software concepts
* Learn Bash scripting
* Automate system operations
* Gain hands-on experience with Linux
* Use GitHub for version control

---

## 🛠️ Tools & Technologies

* Ubuntu (WSL)
* Bash Shell
* Git & GitHub
* Linux Commands

## 📜 Scripts Overview


### 🔹 1. CPU & Memory Monitor

Displays CPU usage and memory consumption.

### 🔹 2. Active User Tracker

Shows currently logged-in users and their count.

### 🔹 3. File Backup Script

Creates compressed backups of directories.

### 🔹 4. Process Checker

Checks if a process is running.

### 🔹 5. File Permission Checker

Checks read, write, and execute permissions of a file.
